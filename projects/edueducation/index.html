项目名称

EduExtension-HIT

运行条件

目前还在开发中，这里列出当前配置

  • Vue3相关组件安装
  • Django相关框架组件安装
  • Python3.9以上
  • LangChain,Spacy,Faiss,LTP相关组件安装

运行说明

当前只能以开发者模式运行

  • 进入后端目录，运行python manage.py runserver
  • 进入前端目录，运行npm run serve
  • 浏览器访问http://localhost:8080
  • 管理员账号：admin， 密码：0
